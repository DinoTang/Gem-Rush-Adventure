# Gem Rush Adventure

A match-3 puzzle game combined with real-time combat mechanics built in Unity.

## 🎮 Gameplay
- Match gems to generate energy
- Monsters automatically attack using energy
- Defeat enemies through strategic matching

## 🧠 Features
- Match-3 core system (swap, match, cascade)
- Energy-based combat system
- Multiple monsters with unique roles
- Level progression (25 levels planned)

## 🛠️ Tech Stack
- Unity Engine
- C#

## 📂 Project Structure
Assets/
- Scripts/
- Prefabs/
- Art/
- Scenes/

## 🚀 How to Run
1. Open project with Unity Hub
2. Open MainScene
3. Press Play
